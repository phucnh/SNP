SNP-SSP
====================

Sample SSP system.

# Prerequisite
- JDK 1.8 (`brew cask install java`) 
- gradle (`brew install gradle`)

# Building
Ensure that your java verison is 1.8

```
$ java -version
java version "1.8.0_45"
```

Build command

```
$ gradle build
```

# Running

```
gradle bootRun
```
